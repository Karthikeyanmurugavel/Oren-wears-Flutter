## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- An IDE (VS Code, Android Studio, or IntelliJ) with Flutter plugins installed.
- An API Key for Google Gemini (if you want to use the AI features).

### Installation

1. **Clone the repository** (or navigate to the project directory):
   ```bash
   cd oren_wears/my_app
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Set up API Keys:**
   - Create a `.env` file or configure your API key in the appropriate configuration file (check `lib/` for generic AI configuration). *Note: Ensure you don't commit your API keys.*

4. **Run the app:**
   ```bash
   flutter run
   ```
